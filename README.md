# HDSB e-Form Website

A comprehensive web application for managing HR and Finance forms.

## Features

- **HR Forms**: Car Rental and Leave forms.
- **Finance Forms**: Claim forms.
- **Role-Based Access**: Specialized dashboards for HR Admins, Finance Admins, Approvers (HOD/HOS), Security Guards, and Super Admins.
- **Real-time Notifications**: Get instantly notified about form status updates.

## Technologies Used

This project is built with:

- React & TypeScript
- Vite
- Tailwind CSS & shadcn/ui
- React Router & React Query

## Local Development

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.
